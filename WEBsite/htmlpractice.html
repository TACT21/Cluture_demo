HTMLとは?

WEBページの文字や画像などを表示するための言語で、
殆どのWEBサイトはこのHTMLを土台に作られています。
このページもHTMLで作られています。
しかし、HTMLはあくまで土台で、HTMLだけでサイトを作ろうとすると、
白黒で面白みのないものになってしまいます。

そこで使うのがCSSです。
CSSはHTMLと同じプログラミング言語で、ページの背景色やレイアウトそのものを変えることができます。

HTMLの構文

HTMLの構文はこのような感じになっています。 

<!DOCTYPE html>
<html>
 <head>
  <title>HTMLを書いてみよう</title>
  <meta charset="UTF-8">
 </head>
 <body>
  <h1>HTMLを書いてみた</h1>
  <p>これがHTMLだ!</p>
 </body>
</html>

見た目は複雑そうですが、中身は結構簡単なのでサクッと覚えていきましょう。
HTMLは、要素名（文字や数字でできた塊）を< >で囲んで「タグ」を作り、文章を構成しています。

<h1>のように要素名から始まるタグは開始タグと呼ばれ、
</h1>のように「 / (スラッシュ)」から始まり要素名がくっつくタグは終了タグと呼ばれます。
開始タグと終了タグで囲まれたところを内容といい、
開始タグ、内容、終了タグを含めた全体を要素といいます。
内容には要素それぞれの効果が適用されます。
この場合h1タグには見出しの効果があるので、
内容の部分（HTMLを始めよう）に見出しの効果が適用されます。

代表的なタグ

HTMLには約100種類のタグがあります。ただ、全てのタグをそれぞれ説明すると日が暮れてしまうので、
今回はその中でも代表的なものをいくつか説明していきたいと思います。

<a> ~リンク~

aタグはリンクを貼るときやページ内をジャンプする時に使います。
非常に汎用性が高いです。
hrefという属性名（後で説明します）は結構使うので覚えておいたほうがいいです。
例：
ここをクリック 

<a href="◯◯">ここをクリック</a><!--〇〇の部分にリンクを入れるとそこに飛びます。-->

<p> ~文章・段落~

pタグは文章を書くときや段落を指定する時に使います。
例：
これは例文です。 

<p>これは例文です。</p><!--〇〇の部分が文章として反映されます。-->

<br> ~文章の改行~

brタグは文章を改行する時に使います。
なくてもpタグを使えば文章は書けますが、個人的にはbrタグの方が好みです。
brタグは終了タグがないので注意してください。
例：
これは
例文です。 

<p>
これは<br>
例文です。
</p>
<!--「これは」の後に<br>を使っているので、「これは」の後ろが改行されます。-->

<h1>〜<h6> ~見出し~

h1~h6タグは見出しを作る時に使います。
h◯の中の数が大きくなればなるほど、見出しのサイズは小さくなります。
非常に使いやすいタグです。
例：

これは例文です。

これは例文です。

これは例文です。

これは例文です。

これは例文です。
これは例文です。
<h1>これは例文です。</h1>
<h2>これは例文です。</h2>
<h3>これは例文です。</h3>
<h4>これは例文です。</h4>
<h5>これは例文です。</h5>
<h6>これは例文です。</h6>
<!--h1~h6タグは自動で改行されるので、注意してください。-->

<img> ~写真の貼り付け~

imgタグは写真を貼り付ける時に使います。
aタグで前後を囲むことで写真をクリックしたら〇〇へ飛ぶ…なんてこともできます。
今回は私が適当に作ったChromeのマークの写真を貼ります。
例：

<img src="〇〇" width="150" height="150">
<!--〇〇の中に写真のリンクを貼り付けることで写真が表示されます。widthは横幅、heightは高さを表しています。-->

<b>,<u>,<i> ~文字の装飾あれこれ~

b,u,iタグは文字の装飾をしたい時に使います。
それぞれの役割については以下の通りです。
bタグ…文字を太字にする
uタグ…文字に下線を引く
iタグ…文字を斜体にする
例：
太字
下線
斜体 

<p>
 <b>太字</b><br>
 <u>下線</u><br>
 <i>斜体</i>
</p>
<!--他にも文字を装飾する方法はたくさんあります。-->

<table>,<tr>,<th>,<td> ~表~

tableタグは表を作りたい時に使います。
border属性を指定すると、枠線をつけることができます。
thタグは見出し的なもので、太字で表示されます。
tdタグは内容的なもので、普通に表示されます。
例：

CSS 装飾

図に表すとこんな構造になっています。

「tableという大きなお弁当の中に、trという仕切りがあって、thという主食とtdという副菜が入っている」
という認識で良いと思います。 

<table border="1">
 <tr>
  <th>HTML</th>
  <td>基盤</td>
 </tr>
 <tr>
  <th>CSS</th>
  <td>装飾</td>
 </tr>
</table>

ちなみに、CSSで飾り付けをすると下のように、枠線や背景色を変えることができます。

<!----> ~コメント~

コメントタグは、「表示はさせたくないけど、ちょっとだけ記述しておきたい」という時に使います。
使い方はとっても簡単です。私もよく使っています。
例：
これは例文です。 

<p>
これは例文です。
</p>
<!--表からは見えない-->

この場合、「表からは見えない」の部分が非表示になります。
Javascriptの場合は「//」、CSSの場合は「/**/」など、
使用する言語によってコメントの仕方が異なるので気をつけてください。

<!DOCTYPE html>

このタグはHTML5という言語で書かれていることを示すためのタグです。
これがないとWebページがしっかり表示されないことがあるので注意です。
このタグは終了タグがないので注意してください。

<html>

このタグは、
「ここからここまでがhtmlで書かれているよ」
ということを示すためのタグです。

<head>,<body>

headタグは、Webページの名前や文字の種類の情報を記述するためのタグで、
bodyタグは、実際にWebページに表示する情報を記述するためのタグです。
図に表すとこんな感じです。

<title>

このタグは、Webページのタイトル部分の名前の情報を記述するためのタグです。
結構大切なので覚えておいてください。

<meta>

このタグは、Webサイトの文字の種類の情報を記述するためのタグです。
このタグを記述しないと、文字関係に色々異常が起きてしまうことがあります。
フォントとはまた違うようで、文字化けを防いだりするときに使います。

実践~HTMLを使ってみよう~

これまでの内容を踏まえて、実際に簡単なコードを書いて遊んでみましょう。
と、その前にHTMLをブラウザで表示させる方法について知っておきましょう。

HTMLをブラウザで表示させる方法

HTMLをブラウザで表示させるには、

  • テキストエディター（ここでは Text を使います）
  • ブラウザ（ここでは Google Chrome を使います）

この2つが必須となります。
（余談ですが「Text」というアプリはGoogleの拡張機能でダウンロードすることができます。）

テキストエディターを開く


Textの場合、開くと「untitled 1」というファイルが開かれ、すぐ書けるようになっています。

とりあえず何か書いてみる

以下のコードをコピーして、テキストエディターに貼り付けてみましょう。 

<!DOCTYPE html>
<html>
 <head>
  <title>HTMLを書いてみよう</title>
  <meta charset="UTF-8">
 </head>
 <body>
  <h1>HTMLを書いてみた</h1>
  <p>これがHTMLだ!</p>
 </body>
</html>

貼り付けたら、好きなファイルに名前をつけて保存しましょう。
HTMLファイルとして保存したいので、ファイル名の終わりに「.html」をつけておきます。

ブラウザで表示する

ファイルを開き、先程保存したファイルをドラッグしてブラウザにドロップします。
すると下のように書いた内容が表示されます。

まとめ

  • HTMLは、要素名（文字や数字でできた塊）を< >で囲んで「タグ」を作り、文章を構成している。
  • aタグを使うことでリンクを貼り、他のページやページ内のある場所にジャンプすることができる。
  • pタグを使うことで文章を書くことができる。
  • brタグで文章を改行することができる。
  • h1~6タグで見出しを、b,u,iタグで文章のデコレーションをすることができる。
  • headタグは、Webページの名前や文字の種類の情報を記述することができて、
    bodyタグは、実際にWebページに表示する情報を記述することができる。
